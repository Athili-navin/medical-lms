# ENAMEL ROADS - Medical Learning Management System

Frontend-only Medical LMS with Next.js 15, TypeScript, Tailwind, shadcn/ui, Zustand, React Hook Form, Framer Motion.

## Quick Start

cd medical-lms
npm install
npm run dev

## Demo Login

Student: /login - any email + 6+ char password
Tutor: /login?role=tutor

See src/lib/mock-data for all mock data.
